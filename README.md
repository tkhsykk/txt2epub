# txt2epub
勉強用その2

機能:  
小説家になろうからダウンロードした章ごとのtxtファイルを1つのepubで出力する。  
Python勉強用  
そのうちGUIにしたい。  

___

## 入力ファイル仕様メモ

- 対象フォルダ: `text/[作者名]作品名/`
  - 拡張子: `.txt`
  - 出力は`[作者名]作品名.epub`となる
- 文字コード: UTF-8（BOMなしを推奨）
  - Shift_JIS 等の文字コードには現在対応していません
- ファイルサイズ: 1ファイルあたり目安として 50KB〜1MB 以下
  - EPUBに変換後の閲覧性を考慮して、章ごとに適切な分割を推奨します
